### Getting Started

These instructions will get you a copy of the project up and
running on your local machine for development and testing purposes.

### Requirements

- Bun: https://bun.sh/

```bash
brew tap oven-sh/bun
brew install bun
```

### Installing and running the app

- Run server

```bash
./scripts/dev
```

### Build for production

- Build app

```bash
bun run build
```
